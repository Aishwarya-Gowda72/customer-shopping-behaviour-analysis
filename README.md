# Customer Shopping Behaviour Analysis

## Overview

This project presents an end-to-end analysis of customer shopping behaviour using Python, SQL, PostgreSQL, and Power BI. The objective is to transform raw customer data into meaningful business insights through data cleaning, SQL analysis, and interactive visualizations.

## Objectives

- Clean and preprocess customer shopping data
- Perform exploratory data analysis (EDA)
- Analyze customer purchasing behaviour using SQL
- Build an interactive Power BI dashboard
- Generate actionable business insights

## Project Workflow

### 1. Data Collection
- Imported the customer shopping behaviour dataset.

### 2. Data Cleaning & Preprocessing
- Removed duplicate records
- Handled missing values
- Corrected data types
- Standardized column names
- Prepared data for analysis

### 3. Exploratory Data Analysis (EDA)
- Customer distribution
- Purchase amount analysis
- Product category analysis
- Customer rating analysis
- Age group analysis

### 4. SQL Analysis
Business questions answered using PostgreSQL:

- Total revenue generated
- Average purchase amount
- Customer segmentation
- Category-wise sales analysis
- Purchase trends
- Revenue by customer demographics

### 5. Dashboard Development

An interactive Power BI dashboard was created to visualize key business metrics and customer insights.

Dashboard features include:

- Total Customers
- Average Purchase Amount
- Average Customer Rating
- Revenue by Category
- Sales by Category
- Revenue by Age Group
- Sales by Age Group
- Interactive Filters
